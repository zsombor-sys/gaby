GABY'S BUDAPEST DEMO

LOCALHOST
1. Csomagold ki a mappát a Desktopra
2. Nyisd meg VS Code-ban
3. Terminal:
   cd ~/Desktop/gabys-budapest-bilingual-demo
   python3 server.py
4. Nyisd meg:
   http://localhost:3000

GITHUB
1. GitHub repo neve legyen:
   gabys-budapest-bilingual-demo
2. Terminal ugyanebben a mappában:
   git init
   git add .
   git commit -m "first commit"
   git branch -M main
   git remote add origin https://github.com/SAJATNEVED/gabys-budapest-bilingual-demo.git
   git push -u origin main

VERCEL
1. Lépj be Vercelbe
2. New Project
3. Válaszd ki a repót
4. Deploy
